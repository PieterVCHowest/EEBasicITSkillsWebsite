Pieter Van Canneyt

Ik begon met het downloaden van VMware. Daarna selecteerde ik "Create new Virtual Machine".

Vervolgens koos ik voor "Typical Configuration".

Bij de volgende stap koos ik voor "Installer Disc Image File, zodat ik een .iso bestand kon uploaden. Dit was namelijk het bestand die ik via Academic Software had kunnen downloaden zodanig dat ik Windows 10 Pro Edu via dit bestand kon installeren. Zoals je in deze screenshot kan zien selecteerde ik dit bestand om te installeren.

Hier geef ik mijn Windows Product Key in die ik kreeg. Ik koos voor Windows 10 Education, aangezien dit de versie van Windows was die ik kon verkrijgen via Academic Software. Als laatste kan je hier zien dat ik mijn volledige naam en een wachtwoord heb gekozen.

De volgende stap was het geven van een naam aan de Virtuele Machine en de locatie waar de Virtuele Machine moest geïnstalleerd worden.

Vervolgens werd mij gevraagd hoeveel ruimte ik wou voorzien voor mijn Virtuele Machine. Ik koos voor 60GB omdat dat da aangeraden grootte was. Vervolgens koos ik er ook voor om alles als 1 bestand op te slaan.

Nadat ik op "Next" klikte, ging VMware direct over naar de opstart van de Virtuele Machine.

Vervolgens moest ik het besturingssystemen selecteren die ik wou installeren. Ik koos natuurlijk voor de enige optie: Windows 10 Pro Education.

Het systeem werd geïnstalleerd.

Windows werd geconfigureerd.

Toen kreeg ik dit scherm: Windows was geïnstalleerd. Nu kon ik in de Virtuele Machine werken.
